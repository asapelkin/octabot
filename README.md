# octabot
Telegram Bot -- GNU Octave (MATLAB) interpreter prototype

### build & run
```
cd octabot
echo "token = <YOUR TOKEN HERE>" > config.py
docker build . -t octabot
docker run  --restart=always  -d octabot
```
